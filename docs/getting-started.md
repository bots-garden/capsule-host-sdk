## Create a Capsule application

Create a directory `say-hello`

```bash
mkdir say-hello
cd say-hello
```

Initialize a new project in `say-hello`:

```bash
go mod init say-hello
```

Install the Capsule **HDK** dependencies:
```bash
go get github.com/bots-garden/capsule-host-sdk
```
